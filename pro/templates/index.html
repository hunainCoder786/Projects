Website Performance Analyzer

Welcome to the Website Performance Analyzer! This tool helps you assess the performance of any website by measuring its load time, page size, and number of requests. Simply enter a URL, and you'll receive a detailed analysis of its performance.

How It Works: After entering a URL, our application sends a request to the specified website, records the time it takes to load, and gathers data about the resources it requires. You'll receive insights that can help improve website performance.

Tech Stack Used: Python, Flask, and JavaScript
